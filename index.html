⚙️ 手動設定

要用已登入 FB 的瀏覽器開啟 HTML (不用開著 FB)，且要先「1. 關閉下載前詢問位置」，並「2.允許彈出視窗」

1. 輸入"chrome://settings/downloads" 關閉「下載每個檔案前先詢問儲存位置」； 建議暫時把【位置】從「Downloads」變更為 乾淨的資料夾路徑

2. 設定 > 隱私權和安全性 > 網站設定 > 允許彈出式視窗與重新導向：貼上 { html 執行頁面那串 url }

用新瀏覽器或新的 index 路徑執行時，一定要先守在瀏覽器旁邊點選【一律允許】，否則會只下載一部影片

⚠️ 注意事項

執行過程中，在 Chrome 作業的游標會一直被目標瀏覽器搶走，建議同一部電腦先用 Edge 或 Firefox 做事情
